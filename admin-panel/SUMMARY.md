
# 🚀 Ertah Admin Panel - Project Summary

## 📋 Overview
This admin panel is a comprehensive management interface for the **Ertah** home services application. It allows administrators to manage users, providers, orders, products, stores, and system settings, ensuring full control over the platform.

## 🛠 Features & Capabilities

### 1. User & Provider Management
- **Users:** View listed users, edit details, manage wallet balance (deposit/withdraw), view address history, and manage loyalty points.
- **Providers:** Approve/reject provider applications, view attached documents, and manage service assignments.

### 2. Service & Order Management
- **Orders:** Track order status (Pending -> Assigned -> In Progress -> Completed), view details, and assign providers manually if needed.
- **Services:** Manage main service categories (Plumbing, Cleaning, etc.).

### 3. E-Commerce (Store)
- **Stores:** Manage spare part stores.
- **Products:** CRUD operations for products and spare parts.
- **Product Categories:** Organize products into specific categories.

### 4. Marketing & Engagement
- **Banners:** Manage app-wide promotional banners.
- **Offers:** Create and manage special offers.
- **Coupons:** Manage discount codes.
- **Notifications:** Send push notifications to users or providers.
- **Loyalty Program:** Manage rewards and redeemable items.

### 5. Finance & Reporting
- **Transactions:** Full financial log of all wallet transactions (deposits, payments, refunds).
- **Dashboard:** High-level statistics and recent activity.

### 6. System & Content
- **Settings:** Feature Toggles (Enable/Disable SMS, Free Mode Simulation, Maintenance Mode), General Configuration, and Compliance Content.
- **Complaints:** Ticketing system for user inquiries and issues.
- **Reviews:** Monitor and moderate user ratings and reviews.
